== README

2Tr is a web based application to help people to learn how to code through online-based curriculum. 
2Tr provides a video chat based flatform to connect code learners with volunteer tutors. 

2Tr offers:

* Verified Online based curriculums for Ruby, HTML/CSS, Javascript and basic Computer Science

* Tutors can anser any questions

* Student learning status with D3 technology

* Certification to Tutors for their volunteering work

* Login through LinkedIn
